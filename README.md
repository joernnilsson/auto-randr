# auto-randr
```
usage: auto.py [-h] [--setup SETUP] [--dry-run]

Autoconfigure montitor setup

optional arguments:
  -h, --help            show this help message and exit
  --setup SETUP, -s SETUP
                        override setup autoselection, must be one of:
                        [external_on_right_align_bottom, mirror, builtin_only,
                        external_only]
  --dry-run, -d         dry run, only print xrandr command
```
